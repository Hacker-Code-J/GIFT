P_{64}(i)=4\cdot\floor*{\frac{i}{16}}+16\cdot\left[\left(3\cdot\floor*{\frac{i\bmod 16}{4}}+(i\bmod 4)\right)\bmod 4\right] + (i\bmod 4).
\]

\[
x_{P(i)}\gets x_i
